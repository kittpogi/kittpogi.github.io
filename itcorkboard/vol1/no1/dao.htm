Oracle Discoverer 2000: An Evaluation
The office has just completed the evaluation of the Oracle Discoverer 2000. The software is an on-line analytical processing tool (OLAP) that retrieves and analyzes data for decision-making from the existing Oracle databases.

Oracle Discoverer has three (3) basic components, namely: 1) User Edition, 2) Administration Edition and 3) End User Layer. The User Edition is the tool for accessing information from the organization’s database for ad hoc query analysis and reporting. The Administration Edition determines how users will access and view the data. The End User Layer provides the intuitive, business-focused views of the database therefore protecting the end user from the complexities of data access.

Among the features of the software that may be useful for us are as follows:

  1. Creation of calculated items or formulas according to user requirements
  2. Creation of joins and complex folders to join two or more tables in order to derive the required information
  3. Creation of drills that enable the users to navigate to a lesser or greater detail of aggregated information (or summarized sets of figures)
  4. Creation of conditions that enable the administrator to filter the query to the required values only to be retrieved in the user edition
  5. Creation of summary folders to improve query efficiency, with pre-aggregated data that are accessed by user-defined queries
  6. Presence of a Workbook Wizard which guides the user in creating query formats

As an assessment, Oracle Discoverer can serve the ad hoc query and reporting requirements of the University. It has a lot of features and is user-friendly. However, for it to satisfy the information needs of the users, there should be a good understanding of the Oracle database structures, knowledge of the capabilities of the tool, and well-defined business areas. For a tool to be most effective, it must seek to meet the information requirements of the user at the time that these are needed, and in the manner that the information can be understood.
Julio Ceres Ibay, Celine Rivera 02/22/1999
