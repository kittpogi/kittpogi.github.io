Human Resource Data Model

The office, aside from the regular activities of updating system reference files, is finishing the conceptual design of the Human Resource Data Model. It shall start with the design of the Facilities Data Model after completion of the documentation of data gathering and data definition. We have formulated the data gathering questions for the Services Data Model and we hope to start data gathering in the middle of the term.

ISRM Migration

Our Database Administrator has his hands full preparing for the migration of the databases for the Integrated Student Records Management using Oracle. Designs for reports to be generated using Discoverer/2000 will also start this term.

Celine V. Rivera 07/05/1999
