Oracle8i - The Database for
Internet Computing

The product launching of Oracle8I featured the capabilities of the product and presented therein are the tools that cater for application development, content management and application deployment of applications in the Internet.

Oracle8i is for building and deploying Internet and enterprise applications. It would allow data to be managed and stored in centralized servers and allow access of up-to-date information and applications from any computers equipped with a standard web browser.

The Oracle8i include tools such as Oracle JServer, Oracle WebDB, Oracle iFS, and Oracle interMedia.

Oracle JServer, Oracle’s Java VM (Virtual Machine), allow developers to write, store and execute Java codes within the database. This is integrated with the database server resulting to a better performance.

Oracle WebDB is used for building, deploying, and monitoring web database applications. It comes with all the tools necessary to build a dynamic web applications and content-driven web sites. These applications and web sites are contained in the database therefore the only client software needed to develop and run applications is a standard web browser.

Oracle iFS is a Java application that runs on the Oracle8i Java VM. It simplifies storing different types of data in the database. It serves as a single data store containing data for many different applications and provides the data administrator a single system for file storage and messaging.

Oracle interMedia also serves as a data management tool the support rich data types that are used in the Web and other Internet applications. It enables the Oracle8i to manage multimedia contents in the Internet and applications that need access to image, audio, video and text information.

Oracle8i include new features that improve the management and performance of data warehouse applications. The new functionalities include Summary Management, Data Management, Resource Management, Analysis Function, and Data Movement Infrastructure.

Also known as aggregated data, Summary table stores the pre-computed results of aggregated data such as sums, counts, and averages. This feature enhances query performance by computing the query from the aggregated data instead of the detail data. The Summary Management feature is extended through the Query-rewrite, Mechanism for maintaining summary tables and Advisory feature capabilities.

The Data Management features the enhanced partitioning capabilities of Oracle8i. It introduces two new partitioning methods namely the Hash Partitioning and the Composite partitioning.

In Hash partitioning, the administrator simply chooses a partitioning key and the number of partitions. Oracle8i will automatically distribute the data evenly across all partitions. The Composite partitioning is a combination of range and hash partitioning. A table for example, is initially partitioned by range then by hash method.

The Resource Management features Database Resource Management that provides a mechanism for allocating the resources of the data warehouse among the users. An example would be controlling the use of CPU resources by allocating certain percentages of CPU resource (e.g. 60%) to an office while assigning the remaining 40% to other users. Controls can also be exercised in terms of specifying limits in the total number of active sessions and the degree of parallelism of individual queries for each department.

The Analysis Functions are extended analytical capabilities of Oracle8i that provide benefits such as simplified programming, reduced client processing loads, reduced network traffic load and caching aggregations.

The Data Movement infrastructure features transportable tablespace that allows fast mechanism for bulk data-movement from one Oracle database to another.

Most of these added features for data warehouse applications were developed and enhanced from earlier versions of Oracle database as well as other Oracle data warehousing software e.g. Discoverer. Some of the features of Discoverer is incorporated in the Oracle8i. This makes it a powerful tool not only for transactional functionalities but also for analytical purposes.

The Oracle8i tools are said to be the first complete package there is available in the market today for database and Internet application development tool. It would be interesting to know how well it will perform or how big a part it can play in the use of Internet in business or the role it play in the now popular e-commerce.

Julio Ceres R. Ibay 07/05/1999
