HOW TO CUSTOMIZE THE DATE FORMAT IN EXCEL

DATES can be written in the different types of formats. Microsoft Excel has the Number tab, inside the Format-Cells box to enable you to create various types of Date Formats.

You can modify how dates will be displayed using two methods:

 

A. Using the Built-in Date Formats

1. Click on the FORMAT menu, and choose the CELLS option.

 

2. Make sure NUMBER tab is active (as shown below).

3. In the Category list, click on DATE.

 

4. In the TYPE box, choose the appropriate date format.

5. Click on OK button.

 

If there is no four-digit year format, use the method below.

 

B. Customizing the Date Format

  1. Follow steps 1 and 2 above.

  2. In the Category box, click CUSTOM.




  3. Type the date format code required in the TYPE box, using one of the existing codes as starting point.
    Examples:

    To display

    09/17/1999

    09-Sep-1999

    Sep-1999

    		 Use this code

    mm/dd/yyyy

    dd-mmm-yyyy

    mmm-yyyy

     

  4. Click on OK button.

Because the rules that govern the way that any calculation program interprets dates are complex, you should be as specific as possible about dates whenever you enter them. This will produce the highest level of accuracy in your date calculations.
