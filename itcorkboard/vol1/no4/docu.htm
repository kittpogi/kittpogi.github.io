DLSU Entrance Test On-line Query (Year 2)

The DLSU-Manila Entrance Test results for School Year 2000-2001 were made available on the DLSU-Manila and DLS-PSI Web Pages last January 3, 2000.  Applicants who were eager to know their status inquired by keying-in their first name and surname.  If the applicant’s name is found, his/her name, chosen degree program, and high school name/location are displayed.  Otherwise, the message “not found” is displayed.  The file does not include the results of STUFAP applicants, which will be released by late January 2000 via postal mail.  The Admissions Office has also released notification letters to all applicants.

Electronic Class list 

As part of the Web Development Team's goal to make the DLSU Intranet a vital tool in the University's customary business activities, the downloading of the Class list via the Intranet was conceptualized and accomplished.

Before, an announcement is e-mailed to all the faculty members advising them the "whens" and "hows" of acquiring a class list.  This involves sending of a registration notice by the faculty member to the specified e-mail address within the specified duration.  ISTC then sends the class list only to those who registered.  But with the electronic class list, no registration is necessary, ISTC just announces the availability on the Intranet of the class list on the 5th week of the term which remains posted for the whole term.  Then, any faculty member who has a NetWare6 account can download the class list in a matter of few clicks on the mouse, anytime within the term.

It can be viewed or printed as a text file or imported as an MS Excel file ideal for plotting the student's grades.  The Web Development Team made it sure that downloading the Electronic Class list will be as easy as 1-2-3 by putting a step-by-step guide within the page. 

Spruced-up SPS Site

After a long time of being oblivious of the changes that has transpired with its neighboring DLSU sites, the SPS web page has now been made over to compliment with the other DLSU web pages.  Information was updated, dead links were reconnected and, similar to other DLSU sites, page layout was standardized. It has been three years, more or less, since the page was refurbished.

The SPS home page is a one-stop-shop site for students to guide them in developing their full potentials as an individual.

Joie Alvarez, 02/11/2000
