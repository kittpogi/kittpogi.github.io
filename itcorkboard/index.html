ITCorkboard

The ITCorkboard was the then Information Technology Center's online newsletter which ran every trimester from Feb 1999 until Mar 2005.

The archive presented here aims to preserve all the issues (warts and all) for posterity's sake, like a time capsule of sorts, and hopefully inspire every one about their work however mundane they think it is. The original code has been preserved as well, all the changes that have been made are just to fix the broken links. However, not all links are working due to the ravages of time (remember Shockwave Flash?) or the batch search + replace fix that I ran didn't catch them.

You can see how dated the designs were, how inefficient we used to code web pages then, and the cutting edge technologies that were featured are several generations obsolete now.

I can't help but laugh and shake my head how foolish we've been, but then I realized that the way we did it was close to state of the art at the time, using the latest tools. During the entire run we have used Adobe Photoshop 5.5 (with ImageReady!) until the first version of the Creative Suite (CS). We used to code using Notepad or HTML Assistant (limited to 32kb HTML files), moved to MS FrontPage, then to Macromedia Dreamweaver Ultradev 4, then to MX.

You will also see occassional typo/misspellings and grammatical errors which made me think if we really did do our editorial job well during that time, or just copied and pasted everything. But one thing's for sure - every issue has been a labor of love, and the contributors from each of the offices were very keen in sharing all the work they've been doing.

And of course, this preface won't be complete without mentioning the used to be password-protected section, Etcetera - the home of ITChika. ITChika is where the Chief Bubwit unleashes the latest gossip, its sharp wit, and know-it-all attitude.

I hope that the first time readers will enjoy this like we did then, and for the "oldies" to reminisce about the good old days and look how we were several pounds lighter in the old pictures.
