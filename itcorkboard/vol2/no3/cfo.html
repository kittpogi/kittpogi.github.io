Computer Facilities and Operation : Updates

WebCT Upgrade
by: Nick Bonao

CFO is currently upgrading WebCT 1.3.1 to version 3.1, this is to give the users more functionality and ease of use.

Laboratory Renovation
by: Nick Bonao

Installation Interactive Response System in M-306 is ongoing, this improvement will make M-306 the most sophisticated and advanced multimedia laboratory in the campus. On the other hand, L-335 is being upgraded from a mere typing laboratory to a multimedia laboratory by equipping it with 47 units of brand new Pentium III with 750 MHz processor.

Network Expansion Plans
by: Nick Bonao

The PC/Network group has already completed the network expansion plan for SPS, Registrar's Office, Psychology, and Biology departments employing modular network management. The group is now working on the expansion plan for Gokongwei building.

Help Desk System (Clientele Customization)
by: Nick Bonao

The development of the Asset Management module of the Clientele (a help desk system) is also ongoing. Once completed, this module is expected to facilitate monitoring and inventory of IT equipment. This project is being done hand-in-hand with SMO.

Network Facilities Improvement (Satellite Dish Upgrade)
by: Nick Bonao

CFO is also replacing the Satellite Dish from 2.4 meters in size to 3.7 meters to increase the down link bandwidth via satellite. The unit is awaiting delivery within the 2nd week of February 2001.
