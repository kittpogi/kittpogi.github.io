New Course Coding System Implemented

A new Course Coding System was implemented for the graduate courses during the enrollment for 1st term, School year 2000-2001.

The system consists of a combination of letters and numeric sequence.  The first three (3) letters of the code represent the college, department or the program that offers the course.  The next three (3) digits represent the course type, e.g., 500-550 are Basic and Foundations Courses for Masters program; 551-559 are Basic Courses for the Doctoral programs; 600-799 are Major Courses and Elective Courses for both level; 800-849 are for Seminar, Practicum, Internship, Independent Studies, or Directed Research for Doctoral programs; 950 are Residency courses for Doctoral program; and 951-999 for Dissertation Writing courses. The last character of the new course code indicates whether the course is Masteral or Doctoral level.

In the new coding system, the different levels, types, departments and programs are easily identified within the course code.  So no more course codes like XCAMOTE, XSETLOG, XTMSYTD.
