Create Custom Shows Within a Presentation in PowerPoint 97/2000

In PowerPoint® 97/2000, you can adapt a single presentation to a variety of audiences by creating custom shows within it. This saves you the effort of duplicating a whole presentation, saving it as a new file, and modifying it to suit a new audience. Custom shows are shows within your presentation, created from slides that you group together. You can group slides into as many different custom shows as you like. You name each custom show, and when you give the presentation, you select the custom show that you want for a particular audience. In this article, you'll learn how to create two types of custom shows within a presentation:

  • Independent custom shows, in which you create several shows based on different slide groupings and show these independently of each other.
  • Linked custom shows, in which you set up one main custom show and use hyperlinks within it to connect to supporting custom shows.

Create Independent Custom Shows

The simplest way to use custom shows is to group the slides in your presentation in various ways and create a custom show for each group. For example, in a presentation about company goals for the new year, you might want to emphasize different goals according to company division. For the Marketing division, you could create a custom show that included only the slides you want Marketing to see - these might be slides 1, 2, and 4. For the Testing division, slides 1, 3, and 4 may be more appropriate, so you'd create a second custom show with these slides. When presenting the show to one of the divisions, you would select the appropriate custom show (Marketing or Testing) for that audience.

The custom shows are all contained within one presentation file, but they run independently of each other; you just select the custom show you want to present.

To create independent custom shows

  1. Open a presentation in which you want to create custom shows. The presentation should include all the slides you might possibly show.
  2. On the Slide Show menu, click Custom Shows, and then click New.
  3. In the Define Custom Show dialog box, in the Slides in presentation list, select the slides that you want for the custom show, and then click Add. For example, if this is the show for the Marketing division, select only the slides that you want that audience to see. To select multiple slides, hold down CTRL as you click the slides.
  4. To change the order in which slides appear in the show, select a slide in the Slides in custom show list, and then click one of the arrows to move the slide up or down in the list.
  5. In the Slide show name box, type a name for the show, and then click OK.
  6. In the Custom Shows dialog box, click Show to preview the custom show, and click through the slides. You should see only the slides for the custom show you just created.
  7. Exit the slide show to return to the default PowerPoint® view. (In PowerPoint 97, you must click Close in the Custom Shows dialog box.)
  8. To create another custom show with a new group of slides, repeat steps 2 through 5, and then click Close in the Custom Shows dialog box.

To set up and view one of the custom shows

  1. On the Slide Show menu, click Set Up Show. Note If the Set Up Show command is not available, check to be sure that your presentation isn't open in slide show view in another window. If it is open, close it; the command is then available in the default PowerPoint view.
  2. Under Slides, click the Custom show option, and select the custom show that you want.
  3. Click OK.
  4. To view the show, click the Slide Show button on the lower left of the PowerPoint window, and then click through the custom show. Note If your slides are not advancing the way you expect them to, display the Set Up Show dialog box (Slide Show menu, Set Up Show) and be sure the correct custom show is selected under Slides. Also check options under Show Type and Advance Slides to see that the settings are the ones you want.

Create Linked Custom Shows

By linking custom shows, you can attach supporting shows to a main show and display the supporting shows only when the audience is interested. For example, in a presentation about a company's new product line, one audience might want more information about Product A, while another audience cares about Product B. Using custom shows, you can create a supporting show of slides that detail Product A, and you can show them just to the Product A audience. Create another custom show with details about Product B, and reserve this supporting show for the Product B audience. Each supporting show links from your main set of slides, and after the supporting show, you click the last slide to return to the main show.

To create linked custom shows, think of your slides in terms of those that will be in the main group and those that will be in supporting groups. First, you create a custom show that includes slides in the main group, then you create separate custom shows for each supporting group. The slides that you want all your audiences to see are grouped into the main custom show. The slides that you want only specific audiences to see are grouped into supporting custom shows. You connect to the supporting shows through hyperlinks that you create in the main show.

To create the main custom show

  1. Open a presentation in which you want to create custom shows. The presentation should include all the slides you might possibly show - slides for the main show and slides for supporting shows.
  2. On the Slide Show menu, click Custom Shows, and then click New.
  3. In the Define Custom Show dialog box, in the Slides in presentation list, select the slides that you want for the main show, and then click Add. These slides should be ones you want all audiences for this presentation to see. To select multiple slides, hold down CTRL as you click the slides.
  4. To change the order in which slides appear in the show, select a slide in the Slides in custom show list, and then click one of the arrows to move the slide up or down in the list.
  5. In the Slide show name box, type a name, such as "Main Show," for the show.
  6. Click OK.
  7. In the Custom Shows dialog box, click Show to preview the custom show, and click through the slides. You should see only the slides for the custom show you just created.
  8. 8. Exit the slide show to return to the default PowerPoint® view. (In PowerPoint 97, you must click Close in the Custom Shows dialog box.)

To create supporting custom shows

To create each supporting custom show, follow the steps in the above procedure starting with step 2. In steps 3 and 4, instead of slides and a name for the main show, select slides for the supporting show - just those slides that you want for a specific audience - and give the show a name that helps you recognize it as a supporting show. If the show gives details about Product A, for example, you could call it "Product A Details." 

To link to a supporting show

  1. Display the slide in which you want to create a link to a supporting custom show. This will be a slide in your presentation that you have made part of your main custom show. In the example of the products-line slide presentation, this would be the slide that lists all the products: Product A, Product B, and so on.
  2. Select the text or object you want to be the link to the supporting show. This might be a product name in the list, such as Product A.
  3. On the Slide Show menu, click Action Settings.
  4. Select the Hyperlink to option, and then select Custom Show from the list.
  5. In the Link To Custom Show dialog box, select the name of the supporting custom show, such as "Product A Details," that you want to link to.
  6. Select the Show and return check box. Selecting this check box automatically links the last slide in the supporting show back to the slide you originally linked from in the main show.
  7. Click OK.
  8. In the Action Settings dialog box, click OK.

To link to additional supporting shows, repeat the above steps.

Tips

  • When you create a hyperlink in the Action Settings dialog box, you can also select options that will add a sound and highlight to the hyperlink when you click it or point to it. Select the options you want on either the Mouse Click or Mouse Over tab. (Highlights apply to objects only.)
  • If you don't want a hyperlink in a custom show to be obvious to your audience, you can apply a shape to the text, create the hyperlink on the shape, and then "hide" the shape by deleting its fill and border. First, click the Rectangle button on the Drawing toolbar, and draw a rectangle to cover the text. Then, select the rectangle and create the hyperlink in the Action Settings dialog box (steps 3 through 8, above). On the Format menu, click AutoShape, and under Fill, click the Color arrow and select No fill. Under Line, click the Color arrow and select No line, then click OK. When you run the slide show, you'll see the text but no evidence of a hyperlink.

To set up the presentation

  1. On the Slide Show menu, click Set Up Show. Note If the Set Up Show command is not available, check to be sure that your presentation isn't open in slide show view in another window. If it is open, close it; the command is then available in the default PowerPoint view.
  2. Under Slides, click the Custom show option, and select the main custom show from the list. This designates your main custom show as the group of slides to run for this presentation.
  3. Click OK.

To view the presentation

  1. In the lower left of the PowerPoint window, click the Slide Show button, and click through the presentation until you reach a hyperlink for a supporting custom show.
  2. Click the hyperlink, and then click through the supporting custom show slides. When you click the last slide of the supporting show, you are returned to the main show and the slide you linked from.
  3. Click additional hyperlinks, if you have them, in the main show to see more supporting shows, and then click through the slides in your main show.
