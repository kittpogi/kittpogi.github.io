    DAO Updates
 

Posting of School and Degree List on the Net

The School and generic Degree listing are now on the Intranet.  This is to get rid of the long printout of the school and degree masterlists which get easily outdated when new entries are added.  With the listings on the intranet, updating is easier and printing cost is eliminated, more importantly, the updated information is readily available to the users.  The Registrar and Admissions Offices use these listings for their reference.  

 

New SRF format for School Code Updates

School updates can now be requested through email using the update form schoolsrf.dot. The users just need to type the new schools on the update form, save it as another file, then send the renamed file as attachment to cscjcri@dlsu.edu.ph. When a school code has been assigned, the form/file is emailed back to the user with the assigned school code or other comments in case the request cannot be granted.  The objective of this new procedure is to facilitate the updating process of the school masterlist. Users have already been notified of this new process, and attached in the notice is the dot file template.
