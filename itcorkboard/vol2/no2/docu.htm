Getting in Gear for the Webby

After a disappointing loss from last year's Philippine Webby Awards, the web development team unveiled the new DLSU Web Site, which we hope this time will get the judges' and the voters' nods. Sporting a brand new look and a flashy Shockwave movie, the new design is in tune with the marketing campaign of the University "Think it. Live it. Believe it."

Commencing from the preliminary design studies until the new site's online "debut", it took the web development team three months to revamp the site. And it's not only on the outside the team did changes. The site now resides on a Win2K/Internet Information Server which have great new features for site management and security.

Currently the team is busying themselves on the new DLS System and DLS-PSI web sites and in adding new features to the DLSU-Manila site like the search engine, forms processing, 3D campus map, etc.

Breaking News!!! DLSU-Manila was selected as one of the semi-finalists for the 2000 Philippine Webby Awards. Visit the official site for more information.

DLSU Undergraduate Application Forms Now Online

The Information Technology Center in coordination with the Admissions Office and STUFAP Office added downloadable DLSU Undergraduate Application Form and STUFAP Application Form on the DLSU-Manila web site. This is good particularly to those people who live far from DLSU or for those who can't stand the long queues at the Admissions Office. For more information, visit the Admissions Office page or click here.

Class Suspension Announcements

Due to the difficulty of the Security Office staff in implementing the present system of announcing class suspension via the web, it will be changed into a more user-friendly interface with no more obscure syntaxes. Presently, the Security Office need to run a batch file using DOS-based commands that they can't seem to remember. Maybe, they lack practice in using it but that's another story. The new system that was proposed will sport a user-friendly GUI (graphical user interface) where they will simply fill out the fields for the date, time, and level (e.g., undergraduate, graduate) of the suspension.

Francis Mark D. Santos 10/16/2000

 

Web Publishing Standards, et al

The Documentation Office has come up with standards on web authoring and publication.  These include:

  • Policies on Web Authoring and Publishing
  • Guidelines for Maintenance of Web Pages
  • Checklist for Creating Web Pages

The policies will guide department/units in requesting web development services from ITC.  These cover requirements in web development and updating, policy on the use of the standard template, policy on accessing web files, policy on official DLSU web pages, policy on posting forms (electronic or downloadable) on the web and other web-related matters.  Details of these policies are posted on the Intranet (http://intranet.dlsu.edu.ph/policy/itc/web_authoring.htm).

The guidelines on web maintenance provide information for department/units maintaining their own web sites.  This is to ensure conformity to the standard template and ensure consistency in the DLSU web design.  The guideline is also posted on the Intranet (http://intranet.dlsu.edu.ph/policy/itc/web_maintenance.htm).

The checklist gives a procedure on how the ITC implements requests for web site development.  The requests may come from DLSU offices which have no existing home page in the DLSU web site.  The checklist is on the Intranet (http://intranet.dlsu.edu.ph/policy/itc/web_checklist.htm).

 

Celine V. Rivera 10/18/2000
