    CFO Updates
 
  • Migration of mail server from Pegasus to Outlook 2000 version was started. Using Outlook 2000 (mail utility from Microsoft) is expected to give us more e-mail functionality.

  • COMPAQ Proliant ML570 series using SAP was acquired for CCS students.  Windows 2000 server operating system and SAP, and COMPAQ Proliant 8000 series for Faculty Mail Server were likewise installed.

  • Backup Tape Library for ITC-CFO was acquired to have a centralized back up system.

  • Additional Xylan switch is being acquired to improve the network traffic in the campus.

 
  • Additional telephone lines for dial-up connection using 3COM Total Control Remote Access Server were acquired to give our students and faculty more room in accessing the internet.

  • De La Salle University was considered by Microsoft Philippines to set-up a Microsoft Exchange 2000 Pilot Program with the help of  Nexus Technologies, Inc.  DLSU will test and run the new Microsoft Exchange 2000 server (mail server facility).  This is called a pilot program for Exchange 2000 server.  The benefits of this are the following:
    Support from Nexus :
    • Three-day on-site support to do planning, design and start-up services;
    • Two-day on-site deployment;
    • Five-day on-call support during the evaluation;

    Support from Microsoft Philippines:

    • Provision of an RTM copy of Microsoft Exchange 2000 server;
    • CTEC Training for one person on Course 1569 Exchange 2000 Administrator.
  • ITC purchased the Clientele Help Desk software.  This software is expected to facilitate tracking of all requests (for repair, etc.) of users, to improve our PC inventory, and to generate adhoc reports.
