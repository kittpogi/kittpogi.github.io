OLS
 By: Ryan Salvanera

The enhanced Online Survey System aims to become user-friendlier to the target market. The new design allows the user to enter their choices on a single page and submit them all at once. This is a far cry from the old design where users had to submit the page for every question, a lengthy process especially if they have a slow connection.

The enhanced system makes use of a single static ASP page connected to a SQL Server 2000 database, which can be dynamically created using the OLS Manager program. This program was developed using VB with a connection to the SQL Server 2000.

The enhanced system is currently undergoing testing. Still, surveys are already lined up. Some of these include those for the ITC (on availability of computer equipment and Internet connections from the students' houses), Caucus on Poverty Reduction (on expertise of faculty), EVP-PSI (on relocation preferences), and AVP-Academic Services (on Computer Loan Application).

GATES on SQL Server 2000
 By: Ryan Salvanera

The Security System at the nine gates around campus has recently been upgraded from a FoxPro application to a VB application. This is in line with the revision of the IDA system, from using a FoxPro database to a SQL Server 2000.

The main feature offered by the new Security System is the ability to display the images of students and faculty.

 

Course Approval System (CAS) for CCS
 By: Arfeli Balanza

A system that will ease the workload of academic assistants (AAs) during enrollment has finally been implemented. The course approval system (CAS) checks for the courses a student has already taken and then evaluates if he/she is allowed to enroll in certain subjects.

With the new system, a student will not be allowed to enlist in an advanced subject if pre-requisites are not yet completed. No more manual flowcharts…no more queues in the AA's offices… no more pre-printed Course Approval Forms (CAPEF), no more tears. He! He! He! Isn't that great?

The lucky college chosen to be the first to use this system was the College of Computer Studies. No wonder, Ms. Divine is all smiles nowadays. Thanks to Rhaniel Bingcang, Arfeli Balanza, and James Sy for this new module of our beloved Student Records Management System.

CAS was initially designed and developed by OJT students Irvin Alzona, Manuel Martinez, Mary Grace Nuguid, and Rodrigo Gajitos, in consultation with Mr. James Sy and Ms. Arfeli Balanza.

Evaluation System (EVA) Up and Running…Finally
 By: Harold Bautista

1999. 2000. 2001. 2002. It sure took its sweet time but, after four years, the evaluation system (EVA) finally managed to stand on its own two feet when the Institutional Testing and Evaluation Office (ITEO) used the system for processing the faculty evaluation results for the 1st Term of SY 2002-2003.

This was the first time the system was used by itself. During the 3rd Term of SY 2001-2002, a parallel run of the old faculty evaluation system (FES) and EVA was conducted. To flush out inconsistencies, the reports generated by EVA were compared with those generated by FES.

Ms. Lorna Acorda coordinated with Ms. Malu Fresado and Ms. Ipat Trinidad of the ITEO to correct the discrepancies encountered.

That's another system-in-development down. Yipee! It's eating time again!

P.S. The system was initially developed by OJT students Rodrigo Gajitos, Irvin Alzona, Manuel Martinez, and Mary Grace Nuguid. Mr. Harold Bautista redesigned the system with Ms. Rowena Santos and Ms. Arfeli Balanza, and OJT students Denny Teodoro, Michelle Chua, Brian Evidente, and Joseph Arrieta developed the new system. The system was further revised by Ms. Lorna Acorda to accommodate new requirements.
