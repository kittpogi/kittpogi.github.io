Office of the Director : Updates
by Joie B. Alvarez

The replacement of the Piltel lease has been completed and Liberty Telecom's facility is now being used. With the replacement, DLSU is saving about P28,000 per month…not to mention Piltel's unreliable service.

I've been…

  • Awaiting results from MLS, Oracle Financials/HR Project, CEM, Data warehouse, PC maintenance, server upgrades, fiber cabling, migration to MS Exchange, installation of wireless network equipment, and some other CFO stuff.
  • Researching for specs for a router to replace Cisco 3640 but apparently, it does not need to be replaced anymore. Fortunately, the purchase has not been initiated yet even though the specifications are already complete.
  • Gathering data for a second ISP to service the DLSU system when DLSU-Dasmarinas and DLSU-Health Sciences Campus connects to DLS-PSI. The best option so far could allow DLS System to save US$3,000 per month, but I still need to study the provider's infrastructure.
  • Coordinating with CFO systems group on the evaluation of cache engines.
  • Coordinating with CSB on the configuration of fax server for DLSU system. CSB already has a set-up and it is a matter of expanding that set-up to serve DLSU system.
  • It is initially intended to have one for Manila and PSI then I thought of expanding CSB's facility and save funds to maybe about P200,000 (guess estimate) per campus.
  • Researching on OpenVMS support for Intel's Itanium.
  • Searching for tools like the search engine for Documentation Office, software deployment for PC/Network, and SMS development tools, among others.
  • Assigning topics in the staff meeting for technical update and TQM orientation

I have yet to coordinate with the EVP offices for the presentation of TQM framework (courtesy of prio) for DLSU.
