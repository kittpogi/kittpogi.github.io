DLSU-Manila Web Site Survey

It is very heart-warming to note that still a good number of our DLSU-Manila web site surfers care enough to take the time to tell us what they think and suggest ways to make it more useful. It came with praises. It came with criticisms.

Of the 20 or so correspondents, most belong to the 16-25 age range. One indicated that she is over 50 years old and commended our graphics, layout and style. We have collated the ones that we have received so far, just to present to you how we are doing so far.

Help Desk Redesign Status
By: Rey Mendoza

The Help Desk web site (http://helpdesk.dlsu.edu.ph/) is undergoing a major facelift and reorganization this term. Keeping up with the “new trend” in the web design community, I am designing the Help Desk website in a new way. The CSS web design way.

CSS stands for cascading style sheet. CSS allows the web designer to create a design which is separate from the content and change the look of the whole web site by modifying the CSS only! With CSS, the web designer can layout a web site without using tables.

The disadvantage of using tables in web design is that it takes time to load in a page. Web browsers make sure that each table downloads as a single entity. As a result, none of the materials put in a table will be displayed until all of its contents are downloaded to the client machine for display.

The use of CSS also speeds up site maintenance by separating the content from the layout, writes device-independent pages that are viewable across all browsers, creates web pages recognizable by search-engine spiders, composes accessible codes that has become legal requirements in a number of countries, allows the users of the web site to chooses among designs a programmer has created and provides a print-friendly version of the web site.

DLSU International Student Guide Website
By: Rey Mendoza,Jr.

The DLSU International Student Website Guide is a reference for aspiring foreign students who wish to study at De La Salle University-Manila.

The web design of the Int’l Student Guide is, for the most part, a replicate of its printed version. The cover design of the printed edition served as the background image of the web site. The Marketing Communication Office (MCO) provided the e-files of the graphics posted in it. In the Philippine link, the corresponding PINAS web site link is found in each section.

The navigation design of the web site has been kept simple. Most information can be accessed in one click or less. The Fluid Dynamics search feature is also displayed in most pages.

The DLSU Int’l Student Guide can be found at http://www.dlsu.edu.ph/students/international/index.asp

VPAR Forms

The Office of the Vice-President for Academics and Research (VPAR) should be commended on maximizing the use of the Intranet’s download section. In the Intranet, the number of downloadable VPAR forms is as much as the number of downloadable ITC forms. Now, people need not keep in their files multiple copies of leave forms, Application for Study Grant, Internet Dial-up Memorandum of Agreement, and the likes. Also, since these are fillable forms, they only need their pens for signing.
