Systems Management Office : Updates

Online Payment Now Up

After more than a year of development and testing, our Online Payments System is finally up. With the current system, our web applications can now support payments online. This facility would eventually be available to the Registrar’s Office online request for documents and later on in the Graduate School of Business’s tuition fee payments.

Oracle Payroll Check Writer

The check writer program of Oracle Financials Payroll module is now operational. This module enables the Accounting Office to print checks and store data relating to this activity, particularly the check number (which was not possible before) of the checks they issue everyday in the accounting database. With this, it will be easier for the Accounting Office personnel to generate reports on the checks they issue and eventually monitor them.

Update SFA

The ProCOBOL program that allows the data entry/update of scholars is now archived. A new program, which was developed using Oracle Developer Forms/Reports is now being used by the Student Financial Assistance Office. An added feature of this is its capability to handle multiple donors for a particular scholar. As of this term, the SFA office has 108 donors and 1266 scholars.

Expanded Library Portal System

The Library Portal System (LPS) is now expanded as it includes other non-DLSU schools in its catalog search. Now on its third phase, LPS provides users a single interface to invoke the library web search of different schools and returns the result in a single integrated environment.

Aside from the online catalogs of DLS System Schools (Manila, CSB, Dasma, HSC, and Zobel), the library holdings of UP Diliman, Ateneo, UST, and UA&P are now accessible from LPS.

Online Submission of Final Grades

The Online Submission of Final Grades is now implemented university-wide. Starting last term, the Registrar’s Office implemented the system for use by all faculty members. This utility provides more convenience to faculty members as they will be able to pass their students’ grades from outside the campus at the same time saving the Registrar’s Office the task of encoding them in the system.

An added feature of the system for the students is that they will be able to view their grades at the My.LaSalle portal once it has been confirmed by the faculty.

Update on Human Resource System

After consulting with some administrators of the DLS System, it was decided that the scope of the Human Resource Information System project will now only include two schools, Manila and the College of Saint Benilde. The project team is now in the middle of the system requirements definition stage and hopes to deploy the project by the third term of this school year.

The New Library System Now Operational

The migration of our current library system (TINLIB) to the new Millennium system has just kicked off. Innovative Interfaces, Inc. personnel are coordinating with library personnel and CFO system administrators in setting up the necessary hardware and software for use in the upcoming training. Initial training is scheduled on July 2 to 4 and will be attended by selected library and SMO staff.
