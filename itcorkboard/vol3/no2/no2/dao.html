Data Administration Office : Updates

Consolidation of archived files
by: Flor Tamonan

We have started the merging of archived files, specifically, data stored in the legacy systems. As of now, both the archived graduate and undergraduate course offering files and data from the oracle database from year 1990 until the 2001 1st term have been consolidated. However, because of missing backup files, we plan to collect data only from year 1995-2001. While we are waiting for the retrieval of the archived course offering files of GSB and CSB, we are already preparing for datawarehousing. Tentatively, we hope to finish the consolidation at the end of this term.

Evaluation of Datawarehousing/Query tools
by: Flor Tamonan

There are three datawarehousing and query tool softwares under evaluation, namely: Seagate Info, SQL Analysis Server and Oracle Discoverer. Testing is ongoing and actual reports are being generated to make sure these provide the features of an effective datawarehousing software and these are capable of giving user-friendly interface, readable and comprehensive information for all levels of management. We envision the implementation of the datawarehousing project on the 3rd term.

Featuring 3 Data Warehousing Tools:
by: Flor Tamonan

TOOL 1: Seagate Info

  • Complete Suite. It has a user-friendly Front-end user interface both in client workstation and in the web with a sophisticated development tool. OLAP capability.
  • Has a simple and flexible system management and administration
  • Can access multiple data sources.
  • Has web-based adhoc query, reporting and analysis tools.
  • Integrates and communicates with ERP packages, relational databases and OLAP servers.

TOOL 2: SQL 2000 Analysis

  • Data Transformation Services (DTS) better known Extract, Transform and Load (ETL) are integrated in the server which are useful in transferring data from one server to another or from one data source to another data source type.
  • Has a rich OLAP capability and easy-to-use development tool.
  • Database management is simple and flexible for data administrator.
  • Has incorporated Help file and online help.
  • Supports the Open Database Connectivity(ODBC).

TOOL 3: Oracle Discoverer

  • Supports the ODBC access to non-Oracle-based databases.
  • Performs adhoc query, reports and datawarehousing functions.

File Integration
by: Flor Tamonan

The DAO in coordination with the Systems Management Office is undertaking the integration of all masterfiles used by the different information systems. It was learned that most of the systems, particularly, microbased systems, have their own masterfiles and data structures are not standardized. With the integration, consistency in the data field names/definition and data structures, definition of a coding standard and completeness will be achieved. These, however, will mean revisions in some of the programs of the SMO systems. Discussion on the integration of the first file, the department file, was completed. SMO is now preparing the inputs for DAO to create the standard department file.

On-line Query
by: Flor Tamonan

Once the Internet Server stabilizes, the completed School and Degree File Query will be available for viewing on the intranet. DAO is planning to incorporate the other reference files like the department files, citizenship etc. on the net on the third term wherein the users can query on the data, when they need it, as they need it.
